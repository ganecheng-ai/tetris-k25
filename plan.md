# 俄罗斯方块游戏开发计划

## 项目概述
使用 Python 开发一个画面精美的俄罗斯方块游戏，支持简体中文界面，操作与经典俄罗斯方块游戏一致。

## 技术选型
- **GUI框架**: Pygame - 跨平台、简单易用、适合游戏开发
- **打包工具**: PyInstaller - 生成独立可执行文件
- **日志系统**: Python logging 模块
- **构建平台**: GitHub Actions (Windows/Linux/macOS)

## 文件结构
```
tetris-k25/
├── src/
│   ├── __init__.py
│   ├── main.py           # 游戏入口
│   ├── game.py           # 游戏核心逻辑
│   ├── tetromino.py      # 方块定义
│   ├── board.py          # 游戏面板
│   ├── renderer.py       # 渲染系统
│   ├── input_handler.py  # 输入处理
│   ├── logger.py         # 日志系统
│   └── config.py         # 配置文件
├── assets/
│   ├── fonts/            # 字体文件
│   └── sounds/           # 音效文件（可选）
├── .github/
│   └── workflows/
│       └── build.yml     # GitHub Actions 配置
├── requirements.txt      # Python依赖
├── tetris.spec          # PyInstaller配置
├── README.md            # 项目说明
├── plan.md              # 本文件
└── logs/                # 日志目录（运行时生成）
```

## 游戏控制
- ← : 左移
- → : 右移
- ↑ : 旋转
- ↓ : 加速下落
- 空格: 直接落下
- P   : 暂停/继续
- R   : 重新开始
- ESC : 退出
